# Fahrkartenautomat (C# Console Application)

## Beschreibung

Dieses Projekt simuliert den Kauf von Fahrkarten über ein Konsolenprogramm.

Der Benutzer wählt eine Tarifzone (1–4) sowie die Art der Fahrkarte (Kind oder Erwachsener). Das Programm berechnet den Preis für jede Fahrkarte und summiert den Gesamtpreis bei mehreren Käufen.

## Funktionen

Eingabe von:

* Tarifzone (1, 2, 3 oder 4)
* Fahrkartenart (<K>ind oder <E>rwachsener)
* Auswahl, ob weitere Fahrkarten gekauft werden sollen

Berechnung von:

* Preis pro Fahrkarte abhängig von Zone und Alter
* Gesamtpreis aller gekauften Fahrkarten
* Anzahl der gekauften Fahrkarten

Ausgabe von:

* Preis pro Auswahl
* Laufender Gesamtpreis
* Endgültiger Gesamtpreis aller Fahrkarten

## Technologien

* C#
* .NET (Console Application)

## Ziel

Ziel des Projekts war es, die Verwendung von Kontrollstrukturen wie `do-while`-Schleifen, `switch`-Anweisungen sowie Benutzereingaben und Validierungen zu üben.
